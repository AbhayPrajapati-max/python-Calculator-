# python-Calculator-
I made Calculator with Python put numbers and operator like +,-,*, / and this calculate and give answer.

Features
-> Addition (+)
->Subtraction (-)
-> Multiplication (*)
-> Division (/)
-> Save History
-> Delete History

# installation 
1. Clone the repository:
2. ''' bash git clone
3. https://github.com/AbhayPrajapati-max/python-Calculator-.git

# Usage
1. Enter the first number
2. Enter Space 
3. Select an operation (+, -, *, /)
4. give space
5. Enter the second number.
6. view the result

# Example
Enter first number: 12
Enter space
Enetr operator (+, -, *, /)
Enter Second Number: 8

Result : 20
